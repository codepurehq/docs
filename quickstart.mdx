# Quickstart

Follow these steps to quickly set up and use Codepure:

***

## 1. Login

Login to your account to access your dashboard.

<Frame>
  <img src="/assets/login2.png" style={{ borderRadius: '0.5rem' }} />
</Frame>

***

## 2. Team Management

Manage who can access your dashboard and what they can do.

* Open Team Management from the side menu.

* Click "Add New".

* Enter the user’s name and email.

* Choose a role: (Admin -  AppSec - Developer).

* Set password.

* Click "Add User".

<Frame>
  <img src="/assets/team_management.PNG" style={{ borderRadius: '0.5rem' }} />
</Frame>

***

## 3. Configure Integrations

You need to connect your account with GitHub, GitLab, or Bitbucket before you can import your projects.

* In the side menu, click Integrations.

* Choose the service you want to connect: GitHub, GitLab, or Bitbucket.

* Enter the Configuration Info.

* Click "Submit Configuration".

<Frame>
  <img src="/assets/gitlab_config1.png" style={{ borderRadius: '0.5rem' }} />
</Frame>

***

## 4. Create a Project

Add your project so you can scan it.

### Option 1: Manual Upload

Create a project manually by uploading your .zip project's file.

* Go to Projects → Create Project.

* Enter the project name and select project language.

* Upload project files.

* Click "Upload File".

<Frame>
  <img src="/assets/manual_project1.png" style={{ borderRadius: '0.5rem' }} />
</Frame>

### Option 2: Repository Integration

Import your project directly from GitHub, GitLab, or Bitbucket. (You must configure integrations first — see Configure Integrations)

* Go to Projects → Import Project.

* Choose your code host: GitHub, GitLab, or Bitbucket.

* Choose the project which you want to import.

* Select project language.

* Click "Clone".

## 5. Start Scanning

Scan your projects to find vulnerabilities.

* Select the project you want to scan.

* Choose the appropriate engine for your project (SAST, SCA, Container Security, Secret Scanning).

<Frame>
  <img src="/assets/start_scanning.PNG" style={{ borderRadius: '0.5rem' }} />
</Frame>

***

## 6. View Scan Results

High-level results will appear in the right-hand sidebar, giving you an overview of the findings.

<Frame>
  <img src="/assets/scan_results.jpg" style={{ borderRadius: '0.5rem' }} />
</Frame>

***

## 7. Review Findings and Fixes

Codepure provides detailed information about each issue, including the vulnerability type, affected code, and proposed fixes. Use this information to better understand and resolve security issues.

<Frame>
  <img src="/assets/findings.PNG" style={{ borderRadius: '0.5rem' }} />
</Frame>

***

## 8. Generate Reports

Create detailed reports of scan results.

* Open Reports from side menu.

* Choose the project for which you want to generate a report.

* Select your preferred security framework.

* Select report format (PDF, WORD).

* Click "GENERATE REPORT".

<Frame>
  <img src="/assets/reports.PNG" style={{ borderRadius: '0.5rem' }} />
</Frame>

***

By following this process, you can integrate Codepure into your development workflow, ensuring secure code and streamlined remediation of vulnerabilities.
