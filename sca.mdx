---
title: 'SCA'
description: 'Understand Software Composition Analysis (SCA) with Codepure.'
---

Codepure's SCA feature scans dependencies in your project to identify vulnerabilities in open-source libraries. It provides real-time alerts for outdated or insecure components, helping you maintain compliance with security standards.

Key benefits:

* Identifies vulnerable dependencies.

* Real-time alerts for outdated libraries.

* Ensures compliance with standards like OWASP and PCI DSS.
