1.  Log in to your **Codepure Dashboard**.
2.  Navigate to **Source Control** > **GitHub**.

### üìù Step 2: Fill in the Integration Form

Complete the form with your GitHub account details:

| Field | Description |
| :--- | :--- |
| **Username** | Your GitHub username. |
| **Access Token** | A [GitHub Personal Access Token](https://github.com/settings/tokens) with appropriate permissions. |
| **GitHub URL** | Use `https://api.github.com` for GitHub.com or your enterprise URL if self-hosted. |
| **Integration Name** | Any friendly name for this integration (e.g., "My GitHub Repo"). |
| **Is Self Hosted** | Toggle this **on** if you‚Äôre using **GitHub Enterprise** (self-hosted GitHub instance). |

---

### ‚úÖ Step 3: Save and Verify

* After filling out the form, click **Connect**.
* Once connected, Codepure will store your integration securely.

---

### üöÄ Step 4: Import Your GitHub Project

Now that your GitHub integration is complete, you can import your repositories.

1.  Go to **Projects** ‚Üí **Import Project**.
2.  Choose your code host: **GitHub**.
3.  Choose the project which you want to import.
4.  Select the **project language** and **branch**.
5.  Click **"Run First Scan"**.
6.  Now you can click **"View Full Report"** to see the full results.
