# 🚀 Codepure Roadmap

We’re building the future of secure software.  
Here’s what’s coming next for Codepure.

---

## 📅 Q4 2025: Major Upgrades

- **🛡️ Full Pipeline Security & Centralized Control**  
  Secure every stage of your DevSecOps pipeline, with centralized management from the Codepure dashboard.

- **🤖 AI-Powered Security**  
  AI engine running fully on-premises to suggest fixes, automate remediation, and reduce false positives.

- **📝 Infrastructure-as-Code (IaC) Security**  
  Detect and fix misconfigurations in IaC templates before they reach production.

- **🔍 Dynamic API Security Testing**  
  Identify vulnerabilities in live APIs and secure your entire API surface.

- **✨ Modern UI/UX Enhancements**  
  A new, faster dashboard with improved navigation and developer workflow.

---

## 🌍 2026: Global Expansion & Next-Gen Features

- **☁️ Codepure as SaaS**  
  Launching Codepure as a secure cloud-based platform for users worldwide.

- **🏢 Multi-Tenancy & Advanced Compliance**  
  Support for multiple organizations, strict data separation, and compliance with major global regulations (PCI, SAMA, NCA, GDPR, and more).

- **🛠️ Automated PR Fixes**  
  One-click auto-remediation: create secure Pull Requests to fix vulnerabilities instantly.

- **🛒 Marketplace Integrations**  
  Codepure available in AWS, Azure, GitHub, and GitLab marketplaces for easy deployment.

- **🔗 SBOM & Supply Chain Security**  
  End-to-end software supply chain protection, including SBOM generation and monitoring.

- **🤝 GenAI Copilot**  
  AI assistant to help with secure coding, documentation, and security best practices.

- **📱 Mobile & Cloud-Native Security**  
  Expanded scanning for Android, iOS, AWS Lambda, Kubernetes, and more.
