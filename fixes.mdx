# Auto Fixing

## Scanning Your Repository

Before applying fixes, ensure you have scanned your repository for vulnerabilities.

---

## Steps to Fix

1. **Select an Issue**: Identify an issue with a ready fix from your scan results.  
2. **Review the Fix**: Check the suggested fix to ensure it aligns with your code's needs.  
3. **Apply the Fix**: Use the Codepure interface or IDE extensions to implement the fix in your project.  

---

## Troubleshooting

- If no fix is available, allow some time for the system to generate one.  
- Provide feedback if a fix doesn't meet expectations to help improve future suggestions.
