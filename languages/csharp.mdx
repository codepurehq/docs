Codepure provides deep, pattern-specific security scanning for a wide range of C# and .NET frameworks, from modern ASP.NET Core to legacy ASP.NET.

## Modern .NET Frameworks

We scan the latest .NET patterns, understanding the specific risks associated with each.

<CardGroup cols={2}>
  <Card title="ASP.NET Core API" icon="server">

  </Card>
  <Card title="ASP.NET Core Razor Pages" icon="file-invoice">

  <Card title="Blazor Server" icon="display">
